## #2

Написать программу, выводящую результаты арифметических операций (+, -, \*, /, %), выполненных над двумя целочисленными переменными.

### Пример

#### Переменные в коде

```c
int a = 26;
int b = 11;
```

#### Вывод в консоли

```
26 + 11 = 37
26 - 11 = 15
26 * 11 = 286
26 / 11 = 2
26 % 11 = 4
```
